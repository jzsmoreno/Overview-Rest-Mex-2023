## English
The data set is located entirely in the file Rest_Mex_Sentiment_Analysis_2023_Train
The data set consists of 251,702 rows (70% of the original data set. The other 30% will be used as a test set), one for each opinion. Each row contains 5 columns:

1. Title: The title that the tourist himself gave to his opinion. Data type: Text
2. Review: The opinion issued by the tourist. Data type: Text
3. Polarity: The label that represents the polarity of the opinion. Data type: [1, 2, 3, 4, 5]
4. Country: The label that represents the country of the visited place. Data type [Mexico, Cuba, Colombia]
5. Type: The label of the type of place of which the opinion is being issued. Data type: [Hotel, Restaurant, Attractive]

The polarity goes from 1, which means the highest degree of dissatisfaction, to 5, which is the highest degree of satisfaction. It can be interpreted in the following way:
1. Very bad
2. bad
3. Neutral
4. good
5. Very good

## Spanish (Español)
El conjunto de datos se encuentra completamente en el archivo Rest_Mex_Sentiment_Analysis_2023_Train
El conjunto de datos consta de 251,702 renglones (el 70 % del conjunto de datos original. El otro 30 % será utilizado como conjunto de prueba), uno por cada opinión. Cada renglón contiene 5 columnas:

1. Title: El título que el propio turista le otorgó a su opinión. Tipo de dato: Texto 
2. Opinion: La opinión emitida por el turista. Tipo de dato: Texto
3. Polarity: La etiqueta que representa la polaridad de la opinión. Tipo de dato: [1, 2 ,3 ,4 ,5]
4. Country: La etiqueta que representa el país que fue visitado. Tipo de dato: [Mexico, Cuba , Colombia]
5. Type: La eqtiqueta del tipo de lugar del cuál se está emitiendo la opinión. Tipo de dato: [Hotel, Restaurant, Attractive] 

La polaridad va de 1 que significa el mayor grado de insatisfacción, hasta 5 que es el mayor grado de satisfacción. Se puede interpetar de la siguiente manera:
1. Muy malo
2. Malo
3. Neutral
4. Bueno
5. Muy bueno
